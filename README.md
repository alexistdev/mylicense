## About Mylicense

Aplikasi pengelolaan user dan license digital, untuk kebutuhan jual beli produk digital, dimana license akan di embeded ke dalam software digital dan dikelola melalui halaman mylicense.

- [Laravel 8.76.2](https://laravel.com/).
- [Template AdminLte](https://adminlte.io/).
